# WebIntro
Intro Webprogrammierung

## Frontend
- JavaScript
- TypeScript
- CSS
- React (mui: https://mui.com/)

`npm -v`
`npm install vite -g`
`npm init vite` -> Name eingeben, React wählen, TypeScirpt wählen

Regelmäßig `npm install` ausführen

Starten mit `npm run dev`

## Backend
- Java
- SpringBoot (https://start.spring.io/)

## Datenbank
- MySQL
- XAMPP
